# Titanic Dataset Data Visualization

## Project Overview

This project visualizes the Titanic dataset using charts and graphs to communicate important insights and trends related to passenger survival.

## Objectives

* Create meaningful visualizations
* Communicate analytical findings
* Identify trends and relationships

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## Visualizations Included

1. Survival Distribution
2. Gender vs Survival
3. Passenger Class vs Survival
4. Average Fare by Survival Status
5. Age Distribution
6. Average Age by Survival Status
7. Fare Box Plot
8. Missing Values Heatmap
9. Correlation Heatmap

## Key Findings

* Gender strongly influenced survival.
* First-class passengers had better survival outcomes.
* Higher fare-paying passengers were more likely to survive.
* Age had a relatively weak effect on survival.
